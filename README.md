# Part 1 - Business Data Cleaning & Excel Reporting

## Problem Summary
The retail sales dataset contained text inconsistencies, missing values, duplicate records, invalid discounts, date issues and business-rule violations. The objective was to create a clean analysis-ready dataset and business reporting outputs.

## Dataset Description
Order-level retail sales data containing customer information, sales transactions, shipping details, discounts, costs and profits.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Data Validation

## Cleaning Steps Performed
- Standardized text fields
- Removed extra spaces
- Corrected capitalization
- Standardized categories
- Validated dates
- Created calculated columns
- Flagged invalid records
- Reviewed duplicate records

## Business Rules Applied
- Missing region filled as Unknown
- Missing ship mode filled as Unknown
- Missing discount treated as 0 where applicable
- Negative discounts flagged
- Excessive discounts flagged
- Invalid shipping dates flagged
- Cancelled and failed orders excluded from completed sales analysis

## Data Quality Issues Found
- Missing values
- Duplicate records
- Invalid discounts
- Date inconsistencies
- Order status issues

## Pivot Reports Created
- Sales and Profit by Region
- Sales and Profit by Category
- Order Count by Ship Mode
- Profit Margin by Segment
- Issue Orders by Region
- Monthly Sales Trend

## Key Insights
- Regional sales performance varies significantly.
- Technology and Furniture contribute major revenue.
- Some records required business-rule correction before analysis.

## Assumptions & Limitations
- Missing discounts treated as zero when appropriate.
- Conflicting duplicate records were flagged for review.
